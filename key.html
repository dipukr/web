Java Keywords
void Used to specify that a method does not have a return value
boolean Used to declare a variable that can hold one of two boolean values, true or false
byte Used to declare a variable that can hold 8-bit integer values or specify return type of a method
char Used to declare a variable that can hold 16-bit unicode chars or specify return type of a method
short Used to declare a variable that can hold 16-bit integer values or specify return type of a method
int Used to declare a variable that can hold 32-bit integer values or specify return type of a method
long Used to declare a variable that can hold 64-bit integer values or specify return type of a method
float Used to declare a variable that can hold 32-bit real values or specify return type of a method
double Used to declare a variable that can hold 63-bit real values or specify return type of a method
if Used to execute a group of statements when a particular condition is true
else Used to execute a group of statements when if condition is false
switch Used to select and execute one of many group of statements
case Used with the switch statements to mark block of statements
default Used to specify the default block of code in a switch statement
while Used to execute a group of statements repeatedly while a particular condition is true
do Used to execute a group of statements repeatedly while a particular condition is true
for Used to execute a group of statements repeatedly while a particular condition is true
break Used to break out of the inner most loop or switch statement
continue Used to skip to next iteration of a loop statement
return Used to return a value and transfer control back to the caller of a method
assert Used to ensure the correctness of any assumptions that have been made in the program
null Used to indicate that a reference variable does not refer to anything
new Used to create a new object
instanceof Used to test whether the object is an instance of the specified class or implements an interface
try Used to define a block of code to be tested for exceptions while it is being executed
catch Used to catch the exceptions generated by try statements, must be used with try block
finally Used to execute a block of code that follows a try block
throw throw is used to explicitly throw an exception, mainly used to throw custom exceptions
throws Used to declare an exception
final Used to declare final variables, final methods or final class
static Used to indicate that a variable or method belongs to class rather than its objects
volatile Used to ensure variable visibility across threads, meaning reads and writes are visible across threads
transient Used to marks a member variable not to be serialized when it is persisted to streams of bytes
strictfp Used to restrict the floating-point calculations to ensure portability
private Used to indicate that a method or field is accessible only in the class in which it is declared
public Used to indicate that a method or field is accessible anywhere in the program
protected Used to indicate that a method or field is accessible only in the same package or its subclasses
package Used to define a package
import Used to bring certain classes or the entire packages into visibility.
enum Use to define a set of named constants
class Used to define a class
this Used to refer to the current object in a method or constructor
super Used to refer to parent class methods or fields
interface Used to define an interface
abstract Used to declare an abstract class or an abstract method
extends Used to indicate that a class is derived from another class or interface
implements Used to implement an interface
synchronized Used to specify the critical sections or methods in multithreaded code
